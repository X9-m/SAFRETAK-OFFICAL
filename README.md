# SAFRETAK 0.1.0

واجهة المسافر لمنصة سفرتك، مبنية بـ React وTypeScript وVite، ومتصلة مباشرة بقاعدة Supabase عبر REST/RPC.

## التشغيل

```bash
npm install
npm run dev
```

## الفحص

```bash
npm run check
```

## إعداد البيئة

انسخ `.env.example` إلى `.env.local` وضع رابط Supabase ومفتاح Publishable فقط. لا يحتاج تطبيق الويب إلى `service_role` أو أي مفتاح خادم.

## الصور

- الصور الأساسية تُنسخ إلى `public/media` أثناء البناء لتجنب الاعتماد على hotlink وقت الاستخدام.
- إذا تعذر تنزيل صورة، تعرض الواجهة SVG محليًا متوافقًا مع نوع الخدمة.
- كل الصور تستخدم مكون `AppImage` لمنع الصور المكسورة وثبات التخطيط.

## الإصدار

الإصدار الحالي: `0.1.0`.
